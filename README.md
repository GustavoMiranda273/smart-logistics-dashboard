# Smart Logistics Dashboard

A cloud-native application for tracking fleet deliveries in real-time.

## Tech Stack
* **Frontend:** HTML5, Bootstrap 5, Jinja2
* **Backend:** Python Flask
* **Cloud:** Google App Engine (Standard)
* **Databases:**
    * Neon PostgreSQL (Relational Data)
    * MongoDB Atlas (Audit Logs)
* **Security:** Firebase Authentication

## Features
* Secure Login System
* Real-time Dashboard Statistics
* REST API for external integrations
* System Health Monitoring (/test-db)

## Setup
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run locally: `python main.py`
